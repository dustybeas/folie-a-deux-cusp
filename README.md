# Folie à Deux as a Cusp Catastrophe in Coupled Active Inference

Simulation code and figures accompanying the manuscript submitted to *Biological Cybernetics*.

## Contents

- `folie_simulations.py` — All three simulations (asymmetric cusp phase diagram, example trajectories, sensitivity analysis)
- `fig1_phase_diagram.png` — Asymmetric cusp phase diagram (Figure 1)
- `fig2_trajectories.png` — Example trajectories (Figure 2)
- `fig3_sensitivity.png` — Sensitivity analysis heatmap (Figure 3)

## Requirements

Python 3.x, numpy, scipy, matplotlib

## Usage

Each Python file runs independently and regenerates its corresponding figure.
